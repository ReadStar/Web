CSS 적용방식 3가지

  1. External CSS - 외부방식
    *.로 스타일 시트를 따로 분리해서 웹문서의
    head태그 내에 link태그로 불러와서 적용.
    여러 웹문서를 통일성 있게 작업가능, 유지보수 용이
  2. Internal CSS - 웹문서 내장 방식
    웹문서의 head태그 내에 style태그로 스타일 내용을 적용.
  3. Inline CSS - 태그 뒤 위치 방식
    웹문서의 body태그 내에 있는 해당 태그 바로 뒤에 style속성으로 스타일 내용을 적용.
    세가지 방식 중에서 우선순위가 가장 높도 특수한상황(블로그 등)에서 쓰임
    유지보수나 통일성 있는 디자인 구성이 힘듦.

CSS 주요 선택자 3가지

  1. tag선택자 - tag그대로 불러와서 적용.
    강제적, 자동적 적용
  2. ID선택자 - tag뒤에 ID를 지정하고 ID값을 불러와서 적용
    선택자로 활용할 때는 ID값 앞에 #을 붙인다.
    ID값은 해당 페이지 내에서 고유하므로 같은 스타일을 다른 요소에 적용 불가 주로 구조적 용도로 쓰임
  3. class선택자 - tag뒤에 class를 지정하고 class 값을 불러와서 적용
    선택자로 활용할 때는 class값 앞에 .을 붙인다
    페이지에서 여러번 쓸 수 있으므로 같은 스타일을 다른 요소에 동시에 적용 가능 주로 디자인적 용도로 쓰임
    (tag하나에 class 값을 2개 이상 조합하여 사용도 가능)
