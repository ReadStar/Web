display의 속성값들

  1. display: inline; inline 요소로 바꿔줌
  2. display: block; block 요소로 바꿔줌
  3. display: inline-block; 두가지 특징을 결합한 요소로 바꿔줌
  4. display: none; 요소를 화면에서 숨김

inline block

  1. inline요소 - 한 줄에 서로 나란히 올 수 있는 요소 위아래 margin 적용 불가, 크기(width/height)적용 불가
  2. block요소 - 한 줄에 나란히 올 수 없는 요소 margin 모두 적용, 크기(width/height) 적용 가능
