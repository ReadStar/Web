  background-image: url("배경이미지 경로");
  배경에 이미지를 불러와서 적용.

background- ... : ; 배경이미지 관련 속성들

  • background-repeat: repeat; 배경이미지 반복 지정
    1. repeat (기본값) 양사방 반복
    2. repeat-x 가로로만(x축) 반복
    3. repeat-y 세로로만(y축) 반복
    4. no-repeat 반복 안함, 한번만 나타내기
  • background-position: left top | 0 0; 배경이미지 위치
    1. 수평 수직 순서로 방향값을 조합하기 (left/center/right | top/center/bottom)
    2. 수평 수직 순서로 직접값을 지정하기 (단위는 px, %, em, cm... etc)
  • background-size: ; 배경이미지 크기
    1. contain 영역에 최대한 이미지의 가로나 세로를 맞춰서 나타내기
      (이미지 하나는 항상 온전히 나오며 반복되거나 빈공간이 생길 수 있음)
    2. cover 이미지 하나로만 영역을 다 덮기
      (이미지가 잘려나갈 수도 있으나 반복되거나 빈공간이 생기지 않음)
    3. 수평 수직 순서로 직접값을 지정하기(단위는 px,%,em ...etc)
  • background-attachment: scroll; 배경이미지 고정여부
    1. scroll (기본값)자신의 영역 스크롤바에는 고정
      브라우저 스크롤바에는 스크롤 됨
    2. fixed 자신의 영역 스크롤바와 브라우저 스크롤바에
      상관없이 항상 고정됨
    3. local 자신의 영역 스크롤바와 브라우저 스크롤바에
      모두 스크롤됨
