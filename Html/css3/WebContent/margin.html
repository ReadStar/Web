margin = border(테두리)의 바깥 여백

margin 값의 갯수별 적용법

  1. margin: n; (바깥 여백의) 양사방이 값 하나로 동일
  2. margin: n n; (바깥 여백의) 상하와 좌우의 순서로 동일
    3. 영역전체를 항상 중앙 정렬할 때
    margin: n auto;로 해서 좌우를 균등분할 하면
    요소가 가운데 오게 할 수 있다
  3. margin: n n n; (바깥 여백의) 위, 좌우 동일, 아래 순서로 동일
  4. margin: n n n n; (바깥 여백의) 위, 우, 아래, 좌 순서로 적용

margin의 방향별 지정 속성

  1. margin-top : n ;(바깥여백) 위쪽
  2. margin-right: n;(바깥여백) 우
  3. margin-bottom : n ;(바깥여백) 아래
  4. margin-left : n;(바깥여백) 좌

box-model의 2가지 타입

box-model은 2가지 타입으로 나뉨 줄이 바뀌는 block 요소와 줄이 바뀌지 않는(한 줄에 나란히 올 수 있는) inline요소

block요소와 inline요소의 차이점
block요소 inline요소
